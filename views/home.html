Recommended Comic

/recommended

Popular Comic

/popular

example : https://facebook.69dev.id/popular

List Comic

/daftar-komik/:page

example : https://facebook.69dev.id/daftar-komik/1

Detail Comic

/detail/[endpoint]

example : https://facebook.69dev.id/detail/solo-leveling

Search Comic

/search?keyword=[keyword]

example : https://facebook.69dev.id/search?keyword=solo

Read Comic

/read/[endpoint]

example : https://facebook.69dev.id/read/solo-leveling-chapter-33-bahasa-indonesia/

Genre

/genre

example : https://facebook.69dev.id/genre

Genre Comic

/genre/[endpoint]
